# 如何将APP安装到手机 - 3种方法

## 方法一：最简单 - 使用Expo Go（推荐新手）

### 步骤：
1. **在手机上安装Expo Go应用**
   - iOS: App Store搜索"Expo Go"
   - Android: Google Play搜索"Expo Go"

2. **在电脑上启动项目**
   ```bash
   cd C:\Users\Administrator\WorkBuddy\Claw\HushangAuntieInventory
   npm install -g expo-cli
   npm install expo
   npx expo start
   ```

3. **用手机扫描二维码**
   - 电脑上会显示一个二维码
   - 用手机的Expo Go应用扫描
   - 应用立即在手机上运行

### 优点：
- ✅ 无需安装Java
- ✅ 无需Android Studio
- ✅ 立即看到效果
- ✅ 支持热重载

---

## 方法二：生成APK文件（永久安装）

### 步骤：
1. **安装Android Studio**
   - 下载：https://developer.android.com/studio
   - 安装时选择"Standard"安装

2. **打开项目**
   - 打开Android Studio
   - 选择"Open an Existing Project"
   - 打开：`HushangAuntieInventory/android` 文件夹

3. **生成APK**
   - 等待Gradle同步完成
   - 点击：Build → Build Bundle(s) / APK(s) → Build APK(s)
   - 等待构建完成

4. **安装到手机**
   - 找到APK文件：`android/app/build/outputs/apk/debug/app-debug.apk`
   - 复制到手机
   - 在手机上点击安装

### 优点：
- ✅ 永久安装，无需电脑
- ✅ 可以分发给员工使用
- ✅ 性能更好

---

## 方法三：使用在线构建（无需电脑配置）

### 步骤：
1. **上传到GitHub**（如果没有账号）
   - 注册GitHub账号
   - 创建新仓库
   - 上传所有项目文件

2. **使用在线构建服务**
   - **推荐：Microsoft App Center** (免费)
     - 访问：https://appcenter.ms
     - 用GitHub登录
     - 创建Android应用
     - 连接GitHub仓库
     - 开始构建

3. **下载APK**
   - 构建完成后下载APK
   - 安装到手机

### 优点：
- ✅ 无需本地安装开发环境
- ✅ 自动构建
- ✅ 可以设置自动更新
